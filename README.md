# Banking Information System

## Project Description
The Banking Information System is a Core Java console application that simulates basic banking operations. It allows users to create an account, deposit money, withdraw money, check their balance, and view account details through a menu-driven interface.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- View account details
- Menu-driven interface
- Object-Oriented Programming concepts

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Switch Case
- Loops
